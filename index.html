Run to Give Toronto 2023

Run to Give Toronto is a virtual charity run that connects you with your community.

Event Description

Our goal is to raise $35,000 for New Start Foundation, which was built to ensure all Canadian youth have timely access to affordable mental health and addiction programs with world-class outcomes.

With your support, we can work towards reducing wait times for treatment, increasing access to care, and providing hope for a better future for Canadian youth struggling with mental illness or substance use disorder.

Registration Opens: April 5th, 2023

Registration Closes: June 5th, 2023

Run Day: June 17th, 2023

Registration Fee: $45 In Person Option - $25 Virtual Option
